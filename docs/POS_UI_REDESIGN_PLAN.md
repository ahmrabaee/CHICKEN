# خطة تحسين واجهة نقطة البيع (POS UI Redesign)

## 1. الوضع الحالي — تحليل

### 1.1 الهيكل الحالي
```
┌─────────────────────────────────────────────────────────────────────┐
│ [نقطة البيع]  [بحث........................]  [باركود] [ميزان]      │
├──────────────────────────────────────┬──────────────────────────────┤
│                                      │  الفاتورة          [مسح الكل] │
│  ┌────┐ ┌────┐ ┌────┐ ┌────┐        ├──────────────────────────────┤
│  │منتج│ │منتج│ │منتج│ │منتج│        │ الزبون: [بحث...]              │
│  └────┘ └────┘ └────┘ └────┘        │ الضريبة: [قالب]               │
│  ┌────┐ ┌────┐ ┌────┐ ┌────┐        │ [نقدي] [آجل] [مختلط]          │
│  │منتج│ │منتج│ │منتج│ │منتج│        ├──────────────────────────────┤
│  └────┘ └────┘ └────┘ └────┘        │ ┌──────────────────────────┐  │
│  ...                                 │ │ صنف 1  │ 1.5 كغ │ ₪ 22   │  │
│  (55% عرض)                           │ │ صنف 2  │ 2.0 كغ │ ₪ 45   │  │
│                                      │ └──────────────────────────┘  │
│                                      │ المجموع | الخصم | الإجمالي    │
│                                      │ [معاينة] [طباعة] [إتمام البيع] │
│                                      │ (45% عرض، min 520px)          │
└──────────────────────────────────────┴──────────────────────────────┘
```

### 1.2 المشاكل
| المشكلة | الوصف |
|---------|-------|
| **توزيع غير متوازن** | السلة تأخذ 45% + حد أدنى 520px، المنتجات قد تشعر بالضيق |
| **منتجات غير واضحة** | شبكة 2–5 أعمدة، قد لا تظهر كل المنتجات بوضوح |
| **سلة مزدحمة** | الزبون + الضريبة + نوع البيع + السلة + الإجمالي + الدفع كلها في عمود واحد |
| **لا يوجد فصل بصري** | أقسام غير محددة بوضوح |
| **تجربة الماسح** | البحث والباركود في نفس الشريط، قد يشتت الانتباه |

## 2. التصميم المقترح — Layout جديد

### 2.1 المبدأ: "منتجات واسعة + سلة مريحة"

```
┌────────────────────────────────────────────────────────────────────────────────┐
│  HEADER (ثابت، رفيع)                                                             │
│  [نقطة البيع]  [بحث سريع]  [باركود]  [ميزان]  [إحصائيات سريعة؟]                 │
├────────────────────────────────────────────────────┬─────────────────────────────┤
│                                                    │  CART PANEL (ثابت)          │
│  PRODUCTS ZONE (مرن، scroll)                       │  ┌─────────────────────────┐│
│  ┌──────────────────────────────────────────────┐  │  │ الفاتورة      [مسح الكل] ││
│  │ [تصفية: الكل | فئة 1 | فئة 2]  (اختياري)   │  │  ├─────────────────────────┤│
│  ├──────────────────────────────────────────────┤  │  │ الزبون (قابل للطي)      ││
│  │  ┌────┐ ┌────┐ ┌────┐ ┌────┐ ┌────┐ ┌────┐  │  │  ├─────────────────────────┤│
│  │  │    │ │    │ │    │ │    │ │    │ │    │  │  │  │ السلة (scroll)          ││
│  │  │منتج│ │منتج│ │منتج│ │منتج│ │منتج│ │منتج│  │  │  │ صنف × كغ = ₪           ││
│  │  │    │ │    │ │    │ │    │ │    │ │    │  │  │  │ ...                     ││
│  │  └────┘ └────┘ └────┘ └────┘ └────┘ └────┘  │  │  ├─────────────────────────┤│
│  │  ┌────┐ ┌────┐ ┌────┐ ┌────┐ ┌────┐ ┌────┐  │  │  │ الإجمالي: ₪ XXX         ││
│  │  │منتج│ │منتج│ │منتج│ │منتج│ │منتج│ │منتج│  │  │  [نقدي][آجل][مختلط]      ││
│  │  └────┘ └────┘ └────┘ └────┘ └────┘ └────┘  │  │  المبلغ المدفوع           ││
│  │  ... (grid أكبر، 6+ أعمدة على شاشات كبيرة)   │  │  [إتمام البيع]            ││
│  └──────────────────────────────────────────────┘  │  └─────────────────────────┘│
│  ~60–65% عرض                                       │  ~35–40% عرض، 380–450px    │
└────────────────────────────────────────────────────┴─────────────────────────────┘
```

### 2.2 نسب العرض المقترحة

| الشاشة | المنتجات | السلة | ملاحظات |
|--------|----------|-------|---------|
| Desktop (1920px) | 65% | 35% (≈670px) | شبكة 6 أعمدة للمنتجات |
| Desktop (1440px) | 62% | 38% (≈550px) | شبكة 5 أعمدة |
| Tablet (1024px) | 55% | 45% (≈460px) | شبكة 4 أعمدة |
| Mobile | 100% (stack) | 100% (أسفل) | تبويبات أو accordion |

## 3. تفصيل الأقسام

### 3.1 Header (شريط علوي)

| العنصر | الوصف |
|--------|-------|
| **العنوان** | "نقطة البيع" مع أيقونة |
| **البحث** | حقل بحث بعرض ثابت (~280px)، placeholder واضح |
| **أزرار الإجراءات** | باركود، ميزان — أيقونات + نص قصير |
| **اختياري** | عداد سريع (عدد الأصناف في السلة، المجموع) |

**تحسينات:**
- ارتفاع ثابت ~56–64px
- خلفية `bg-card` مع `backdrop-blur`
- ظل خفيف للفصل عن المحتوى

### 3.2 منطقة المنتجات

| الهدف | التنفيذ |
|-------|---------|
| **عرض أكبر** | نسبة 60–65% من العرض، شبكة 5–6 أعمدة |
| **بطاقات أوضح** | حجم ثابت، حدود، hover، سعر بارز |
| **عرض الكل** | `overflow-y-auto` مع ارتفاع محسوب |
| **تصفية اختياري** | أزرار فئات سريعة فوق الشبكة |
| **حالة فارغة** | رسالة + أيقونة عند عدم وجود منتجات |

**بطاقة المنتج المقترحة:**
```
┌─────────────────────┐
│  [صورة إن وُجدت]    │
│  اسم المنتج         │
│  ₪ XX.XX / كجم      │
│  [+ إضافة]          │
└─────────────────────┘
```
- ارتفاع ثابت ~120–140px
- `min-h` لضمان تناسق الشبكة

### 3.3 لوحة السلة

**هيكل عمودي:**

1. **رأس السلة** (ثابت)
   - "الفاتورة"
   - زر "مسح الكل"

2. **قسم الزبون** (قابل للطي)
   - بحث الزبون
   - اسم + هاتف (إن لم يُختر زبون)

3. **قسم السلة** (قابل للتمرير)
   - عناصر السلة
   - كل عنصر: اسم، كمية، سعر، إجمالي، أزرار +/–، حذف

4. **قسم الملخص** (ثابت)
   - المجموع الفرعي
   - الخصم %
   - الضريبة (إن وُجدت)
   - **الإجمالي** (بارز)

5. **قسم الدفع** (ثابت)
   - نوع البيع: نقدي / آجل / مختلط
   - طريقة الدفع: نقداً / بطاقة
   - المبلغ المدفوع
   - المتبقي (إن وُجد)

6. **أزرار الإجراءات** (ثابت)
   - معاينة
   - طباعة
   - **إتمام البيع** (زر رئيسي كبير)

**تحسينات:**
- عرض ثابت 380–450px (بدون min كبير)
- حدود واضحة بين الأقسام
- الإجمالي بلون مميز وحجم أكبر
- زر "إتمام البيع" بارز (لون، حجم، ظل)

## 4. تحسينات بصرية

### 4.1 الألوان والتباين
- استخدام `primary` للإجمالي وزر إتمام البيع
- `muted` للنصوص الثانوية
- `destructive` لمسح الكل والحذف
- خلفية خفيفة للمنطقة الرئيسية (`from-slate-50` أو ما يعادلها)

### 4.2 التباعد
- `gap-4` أو `gap-5` بين البطاقات
- `p-6` للمناطق الداخلية
- مسافات واضحة بين أقسام السلة

### 4.3 الظلال والحدود
- `shadow-sm` للبطاقات
- `shadow-lg` أو `shadow-xl` للسلة
- `rounded-xl` أو `rounded-2xl` للزوايا

### 4.4 الحركة
- `transition-all duration-200` للـ hover
- تأثير بسيط عند إضافة صنف للسلة (مثل toast أو highlight)

## 5. الاستجابة (Responsive)

### 5.1 Desktop (≥1280px)
- تخطيط أفقي: منتجات | سلة
- شبكة 5–6 أعمدة للمنتجات
- سلة بعرض ثابت

### 5.2 Tablet (768–1279px)
- تخطيط أفقي مع تقليل أعمدة المنتجات (3–4)
- سلة أضيق (~400px)

### 5.3 Mobile (<768px)
- عمود واحد
- خيار أ: المنتجات أولاً، السلة أسفل (scroll طويل)
- خيار ب: تبويبان "منتجات" | "السلة"
- خيار ج: السلة كـ bottom sheet أو drawer

## 6. ترتيب التنفيذ

### المرحلة 1: إعادة التخطيط
1. تعديل نسب العرض (منتجات 60–65%، سلة 35–40%)
2. تقليل `min-width` للسلة إلى ~380px
3. زيادة أعمدة شبكة المنتجات (5–6 على الشاشات الكبيرة)

### المرحلة 2: تحسين المنتجات
4. تحسين بطاقات المنتجات (حجم، حدود، hover)
5. إضافة تصفية بالفئات (إن وُجدت فئات)
6. تحسين حالة "لا توجد منتجات"

### المرحلة 3: تحسين السلة
7. فصل أقسام السلة (رأس، زبون، عناصر، ملخص، دفع، إجراءات)
8. جعل قسم الزبون قابل للطي
9. تحسين عرض عناصر السلة
10. تمييز الإجمالي وزر إتمام البيع

### المرحلة 4: Header والتفاصيل
11. تحسين الـ Header
12. ضبط الألوان والظلال والتباعد
13. اختبار الاستجابة على أحجام شاشات مختلفة
